# ADD_PERSONA_HUMOR_LAYER

Ты работаешь внутри `protected-agent-kit`.

Нужно добавить в конкретный persona prompt отдельный стильовой слой анти-идолопоклоннического юмора. Это не rewrite всего prompt и не замена основного характера агента. Это локальный patch.

Сначала определи, подходит ли агенту этот слой вообще.

Подходит, если агент:

- работает с критикой, письмом, методологией, формой мышления, стилем, концептуальной ясностью;
- рискует скатиться в картонный пафос, мёртвую серьёзность, методологическое самодовольство или agent cosplay;
- выдерживает редкий короткий undercut без потери рабочей точности.

Не подходит, если это:

- dry guard prompt;
- runtime enforcement prompt;
- stop/approval logic;
- safety-critical procedural prompt;
- prompt, где малое колебание регистра ломает однозначность.

Обязательно используй как источник:

- `04_KITS/persona_fragments/ANTI_IDOLATRY_HUMOR.md`
- `04_KITS/persona_fragments/SUDDEN_UNDERCUT_STYLE.md`
- `04_KITS/persona_fragments/PERSONA_AGENT_STYLE_PATCH.md`

Порядок работы:

1. Сначала кратко определи, подходит ли этому persona-agent такой слой.
2. Если не подходит, так и скажи, с короткой причиной.
3. Если подходит, найди минимальное место вставки.
4. Не переписывай весь prompt, если достаточно локального patch.
5. Не тащи этот слой в guard/runtime enforcement sections.
6. Не делай из агента шутника.
7. Добавляй только редкий короткий антипафосный undercut как калибровку формы.

Требования к вставке:

- юмор не по умолчанию;
- не постоянный тон;
- без GPT-подмигивания;
- без мемности ради мемности;
- без унижения собеседника;
- без стендапной энергии;
- с ясным различением humour vs cynicism;
- после укола агент должен сразу возвращаться к работе.

Что вернуть в ответ:

- verdict: подходит / не подходит;
- место вставки;
- краткий diff summary;
- точный patch fragment или короткую предложенную вставку;
- пояснение, почему слой не ломает исходный persona prompt.

Запрещено:

- переписывать весь persona prompt без необходимости;
- размазывать юмор по всему документу;
- вставлять этот слой в dry guard/runtime prompts;
- заменять серьёзность балаганом.
