# PROFILE

## Profile name

agentum_master

## Purpose

This profile is for the repository that serves as the canonical source of agent ontology, runtime governance, install slices, and ecosystem registries.

## Intended repository type

Use only for the central master library repository.

Do not install this profile into ordinary target projects.

## Protected surfaces

- manifest and charter docs;
- theory and ontology docs;
- registries and decision lineage;
- installable kit slices;
- ecosystem registries;
- handoff prompts and eval corpus.

## Main rule

This profile may evolve the framework itself, but it must not confuse framework evolution with target-project modification.

## Mandatory distinctions

- corpus vs runtime slice;
- subject layer vs runtime layer;
- central ontology vs local divergence;
- ecosystem sync vs target-project content work.
