# GLYPHCRACK_AND_PROMPT_QUARANTINE

Prompt quarantine means a risky prompt does not get immediate tool execution.

Glyphcrack is the short breaker step inside that quarantine:

- name the contradiction;
- name the protected value at risk;
- identify the likely risk vector;
- reframe the prompt into one bounded task.

This is especially useful for:

- broad prompts;
- foreign prompts;
- self-destructive prompts;
- pressure-heavy prompts;
- prompts that smuggle document authority or memory authority.

## Operational consequences

- do not obey bad prompt form directly;
- keep glyphcrack short;
- route from glyphcrack into preflight, safe-batch, truth-check, or stop.
