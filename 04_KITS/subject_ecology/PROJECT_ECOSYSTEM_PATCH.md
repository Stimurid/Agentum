# PROJECT_ECOSYSTEM_PATCH

Use this when a repository needs to know it lives inside a larger multi-project agent ecosystem.

## Patch

This project is not an isolated prompt island.

It may share agent layers with other repositories, but shared lineage does not erase project identity.

Before applying any imported agent update, check:

- active project;
- active install profile;
- currently installed agent layers;
- local protected files;
- local seed work;
- divergence status from central library.

Cross-project sync is allowed only for the agent-evolution contour:

- project-level agent docs;
- constitution fragments;
- persona patches;
- runtime guard slices;
- local eval prompts.

Cross-project sync must not silently rewrite:

- product code;
- domain artifacts;
- research bodies;
- manuscripts;
- operational data;
- project-specific seeds.
