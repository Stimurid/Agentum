# /safe-batch

Convert "stop microtasking" into bounded related work.

Input checklist:

- task cluster
- why tasks belong together
- max touched files
- max touched areas
- protected seed files
- non-goals
- budget class
- rollback path

Output contract:

- bounded batch plan
- explicit non-goals
- acceptance criteria
- stop condition

Stop condition:

- stop if the batch turns into rewrite, cleanup-everything, or cross-project work
