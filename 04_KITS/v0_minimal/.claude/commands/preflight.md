# /preflight

Use before non-trivial or risky action.

Input checklist:

- active project
- working object
- intended files or areas
- protected paths
- neighboring projects
- risk vector
- budget and stdout risk
- seed, deploy, memory, and document-instruction risk

Output contract:

- action or no-action decision
- named risk vector
- scope limit
- rollback path
- required approval if any

Stop condition:

- stop if project identity, authority, or blast radius is unclear
