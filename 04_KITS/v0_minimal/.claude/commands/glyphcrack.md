# /glyphcrack

Break a weak, broad, foreign, or self-destructive prompt before execution.

Input checklist:

- original prompt
- contradiction or ambiguity
- protected value at risk
- active project

Output contract:

- named contradiction
- named protected value
- bounded replacement task
- next route: preflight, safe-batch, truth-check, approval, or stop

Stop condition:

- stop when the prompt is reframed into one bounded task
