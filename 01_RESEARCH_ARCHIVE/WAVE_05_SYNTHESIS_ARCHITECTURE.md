# WAVE_05_SYNTHESIS_ARCHITECTURE

This wave assembled the architecture from earlier findings.

Core synthesis:

- corpus and registries preserve evidence;
- theory explains why patterns exist;
- `v0_minimal` carries the smallest runtime slice;
- profiles adapt that slice to target project identity;
- evals prove the slice against real failure modes;
- handoff prompts transfer the work without archive collapse.

What this wave contributed:

- the repo layer model;
- the decision to keep persona fragments separate from runtime guard;
- the proving-ground requirement before real installation.

Operational consequence:

Architecture must remain modular enough that runtime files do not impersonate the whole repository.
