# WAVE_04F_MEMORY_SURFACE_MODEL

This wave mapped where memory-like influence enters an agent session.

Key takeaways:

- chat carryover, notes, traces, summaries, echo signatures, and recovered prompts all shape behavior;
- memory poisoning can arrive through documents, logs, and prior prompts;
- memory handling needs provenance, scope, status, and usable-for-action discipline.

What this wave contributed:

- memory surface model;
- recovery sufficiency stop;
- the line between recall, anchor, inference, and verification.

Operational consequence:

Memory should close an evidence gap, not become a parallel religion.
