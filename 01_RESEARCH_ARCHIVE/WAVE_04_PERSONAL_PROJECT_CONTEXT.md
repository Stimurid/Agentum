# WAVE_04_PERSONAL_PROJECT_CONTEXT

This wave preserved high-signal personal project context that shaped profile distinctions.

Key takeaways:

- Litops, ModerBober, WhiteCrow, and Quinta are not variants of one generic repo problem;
- each has different protected objects, failure surfaces, and install constraints;
- shared local infrastructure creates bleed risk even before any file is edited.

What this wave contributed:

- the choice to keep target-specific logic in profiles;
- the choice to keep this repository standalone;
- the insistence on active-project identity before action.

Operational consequence:

Personal context is useful only when translated into explicit profile rules.
