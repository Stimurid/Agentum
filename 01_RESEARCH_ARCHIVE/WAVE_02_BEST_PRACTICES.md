# WAVE_02_BEST_PRACTICES

This wave extracted reusable practices from existing agent workflows without blindly canonizing them.

Key takeaways:

- short root contracts beat giant constitutions in live runtime;
- prompt routing must happen before tool execution, not after damage;
- budget, stdout, and repetition are first-class project resources;
- good practice is only portable if it names scope, blast radius, and stop condition.

What this wave contributed:

- compact root manifest principle;
- safe-batch framing;
- explicit budget/stdout discipline;
- honest status taxonomy.

Operational consequence:

A best practice that cannot survive a bounded real task is not yet a kit pattern.
