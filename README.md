# protected-agent-kit

Standalone infrastructure project for protected coding-agent work.

This repository stores:

- research archive;
- source registry;
- pattern registry;
- failure-mode registry;
- portable Claude Code / Codex guard kit;
- install profiles for target projects such as Litops, ModerBober, WhiteCrow, Quinta.

This is not a runtime prompt dump. Project repositories should receive only selected install profiles, not the whole research archive.
